== Description ==

[WebSite](http://www.mobilerevenu.com/) | [Stats](http://stats.mobilerevenu.com/) | [Facebook](http://www.facebook.com/MobileRevenu) | [Twitter](http://twitter.com/MobileRevenu) | [Contact](http://www.mobilerevenu.com/?module=home&action=contact)

Détecte un terminal mobile et vous permet <strong>de le basculer sur une thème mobile</strong>, ou de le rediriger vers un site marchand adapté et optimisé pour ce terminal, vous toucherez alors une <strong>commission à vie sur chaque vente</strong>. 

Vous avez également la possibilité de rediriger uniquement les vieux terminaux afin de conserver sur votre site les smartphones de dernières générations ou encore les tablettes tactiles.

Le plugin vous ajoutera également un bouton dans l'édition de vos articles en mode visuel vous permettant d'intégrer rapidement un lien de promotion nichée où vous le souhaitez dans votre article. Cette fonctionnalité est idéale si vous avez un blog 100% mobile.

Nous vous invitons à jeter un oeil à la section Screenshots pour mieux comprendre.

= English =

<strong>MobileRevenu</strong> detects the presence of a mobile terminal on your blog and allows you, <em>if you want</em>, to redirect it to a suitable site and optimized for this terminal, you will take a <strong>commission on every sale for life</strong>.

You can also redirect the old terminals only to keep on your site the latest generation of smartphones and tablets.

The plugin will also adds a button in the editor of your articles in visual mode that will allow you to quickly integrate a thematic promotion link anywhere you want in your article. This feature is ideal if you have a 100% mobile blog.

We invite you to take a look at the Screenshots section to better understand.

== Frequently Asked Questions ==

= How can I contribute adding my language ? =

* Download [PoEdit](http://www.poedit.net/download.php) (for free)
* Download and open with PoEdit this file : "[mobilerevenu.po](http://plugins.svn.wordpress.org/mobilerevenu/trunk/languages/mobilerevenu.po)"
* Update : Catalogue > Configuration (language, country, translator...)
* Save your opened file as xx_YY.po
cf. [http://codex.wordpress.org/WordPress_in_Your_Language](http://codex.wordpress.org/WordPress_in_Your_Language)
* Translate each sentense & save your file
* Send your files (xx_YY.po and xx_YY.mo) to julien [AT] mobilerevenu.com

We'll add them on the next update for all. Thank you very much for your help.
